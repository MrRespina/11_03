웹 개발, 웹 퍼블리싱(WEB Design과 WEB Programing의 중간 단계)... 지금은 [서비스]가 주도하는 시대. >> Youtube, Netflix ... 서비스를 통해 새로운 가치 창출이 이루어짐. >> 이 서비스들은 WEB에서부터 시작함. WEB : 연결 고리 Windows, mac, Desktop, TABLIT, Smartphone, Android and IPhone 등 이 모두를 이어주는 매개체이다. WORA(Write Once Read Anywhere) HTML(Hyper Text Mark-up Language) HyperText : 링크(Link) Mark-up Language : 태그(tag)를 사용하여 문법을 표기하는 언어 프로그래밍 언어가 아니고 (표현 언어)이다. 웹 사이트 만들때 쓰는 디자인 언어로 인터프리터 방식이다. 컴퓨터 언어 : 컴파일러 vs 인터프리터 - 컴파일러 : 파일 전체를 기계어로 번역해서 사용 Java : (.java 작업 (java 전체를 기계어로 번역-컴파일 ) > .class > 실행) .java에 문제가 있어서 기계어로 번역이 불가능하면 .jar 파일이 생성되지 않음. - 인터프리터 : 한줄씩 기계어로 번역해서 사용 HTML : .html 작업 > 실행 > 한줄씩 기계어로 번역해서 실행해줌. 잘못 만든 상황에서 이미 실행했으면 error는 따로 나오지 않음. 결과는 어떻게 될지 아무도 모름?
